### Features

- [x] `keygen()` - generate a key pair `(ek, dk)`
- [ ] `key_derive(seed)` - generate a key pair from a provided seed
- [ ] `encaps(ek)` - generate a key and ciphertext pair `(key, ciphertext)`
- [ ] `decaps(dk, ciphertext)`- generate the shared key `key`
